# Deploy / Distribute Workflow

This app is designed to be run locally or packaged via Electron (future).

## Verification

1. `npm install`
2. `npm run build`
3. Fix any TS errors.

## Local Run

1. `npm run dev` or run `start_app.bat`.

## Packaging (Future)

To package for distribution:

1. Ensure `electron` and `electron-builder` are configured.
2. Run `npm run electron:build`.
